# Series Renamer

This code is intended to take the contents of a directory, and rename them by series name, season number, and episode number. 

The format should be the following: SeriesName - SXX - EXX

Example Directory 1:

Directory = "[XXXX] SeriesName - S01E01v2", "[XXXX] SeriesName - S01E02v2", "[XXXX] SeriesName - S01E03v2", "[XXXX] SeriesName - S01E04v2", "[XXXX] SeriesName - S01E02v2"

Example Directory 2:

Directory = "Series Name S01E01", "Series Name S01E02", "Series Name S01E03", "Series Name S01E04", "Series Name S01E05"

Example Directory 3:

Directory = "Series Name (YYYY) - S01E01 - Episode Name (1080p BluRay)", "Series Name (YYYY) - S01E02 - Episode Name (1080p BluRay)", "Series Name (YYYY) - S01E03 - Episode Name (1080p BluRay)", "Series Name (YYYY) - S01E04 - Episode Name (1080p BluRay)", "Series Name (YYYY) - S01E05 - Episode Name (1080p BluRay)" 

Planning to introduce more versatility in the future
